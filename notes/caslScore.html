I can name that score in 2 lines

One of the repeating pattern I encounter in working with our pre-sales teams is scoring user input. Besides MAS they use CAS server for scoring. In this scenario there are three use cases:
  1. Model is published to a CAS destination
  2. Model code is datastep code stored in a modelTable(sashdat)
  3. Model code is in an astore
After several such engagements I have created a js library(restaflib) based on restaf that simplifies the code to 2 lines, ignoring the setup code and the code needed for post-processing the results. Sidebar: The restaflib has several methods for common use cases like
